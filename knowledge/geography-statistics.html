地理数据收集

实地调查法：

  • 野外观测：
    • 气象观测：温度、湿度、降水、风向风速
    • 水文观测：水位、流量、水质监测
    • 地质观测：岩石类型、地质构造、矿物成分
    • 生物观测：植被类型、动物分布、生物多样性
  • 问卷调查：
    • 人口调查：年龄结构、职业分布、收入水平
    • 经济调查：产业结构、就业状况、消费水平
    • 社会调查：教育水平、生活质量、文化特征
    • 环境调查：环境质量、污染状况、保护意识
  • 访谈调研：
    • 深度访谈：获取详细信息和深层次认识
    • 专家访谈：获取专业意见和权威观点
    • 焦点小组：了解群体观点和集体意见
    • 参与观察：深入了解当地文化和生活方式

遥感监测法：

  • 卫星遥感：
    • 光学遥感：可见光、近红外、短波红外
    • 雷达遥感：合成孔径雷达、激光雷达
    • 热红外遥感：地表温度、热岛效应
    • 高光谱遥感：精细光谱信息、物质识别
  • 航空遥感：
    • 航空摄影：高分辨率影像获取
    • 无人机遥感：灵活机动、成本较低
    • 机载激光雷达：精确地形测量
    • 航空高光谱：详细光谱信息

统计资料法：

  • 政府统计：
    • 人口普查：人口数量、结构、分布
    • 经济普查：产业结构、经济总量
    • 农业普查：农业生产、农村发展
    • 环境统计：污染排放、环境质量
  • 国际组织数据：
    • 联合国统计：全球发展指标
    • 世界银行数据：经济发展数据
    • 国际货币基金组织：金融经济数据
    • 世界卫生组织：健康统计数据

数据整理方法

数据清洗：

  • 异常值处理：
    • 识别异常值：统计方法、可视化方法
    • 异常值原因：测量误差、录入错误、极端事件
    • 处理方法：删除、替换、保留并标注
    • 影响评估：对分析结果的影响程度
  • 缺失值处理：
    • 缺失类型：完全随机缺失、随机缺失、非随机缺失
    • 处理方法：删除、插值、回归估计、多重插补
    • 插值方法：线性插值、样条插值、克里金插值
    • 质量评估：插值精度、不确定性分析

数据分类：

  • 定性数据分类：
    • 名义分类：无序分类，如土地利用类型
    • 等级分类：有序分类，如地形起伏程度
    • 编码方法：数字编码、字符编码
    • 分类标准：科学性、实用性、可操作性
  • 定量数据分类：
    • 等距分类：相等的类距
    • 等量分类：相等的频数
    • 自然断点：数据分布的自然分界
    • 标准差分类：基于标准差的分类

数据标准化：

  • 量纲标准化：
    • Z-score标准化：均值为0，标准差为1
    • 最大最小值标准化：缩放到[0,1]区间
    • 小数定标标准化：移动小数点位置
    • 单位向量标准化：向量长度为1
  • 时间标准化：
    • 基期标准化：以某年为基期100
    • 环比标准化：与前期比较
    • 同比标准化：与同期比较
    • 移动平均：消除季节性波动

统计分析方法

描述统计：

  • 集中趋势：
    • 算术平均数：所有数值的平均
    • 几何平均数：适用于比率数据
    • 中位数：排序后的中间值
    • 众数：出现频率最高的值
  • 离散程度：
    • 极差：最大值与最小值的差
    • 四分位距：第三四分位数与第一四分位数的差
    • 方差：数据与均值偏差的平方和的平均
    • 标准差：方差的平方根
    • 变异系数：标准差与均值的比值
  • 分布形状：
    • 偏度：分布的对称性
    • 峰度：分布的尖锐程度
    • 正态性检验：K-S检验、S-W检验

相关分析：

  • 线性相关：
    • 皮尔逊相关系数：衡量线性相关强度
    • 相关系数的意义：-1到1之间，绝对值越大相关性越强
    • 显著性检验：判断相关性是否显著
    • 相关矩阵：多变量间的相关关系
  • 非线性相关：
    • 斯皮尔曼等级相关：适用于等级数据
    • 肯德尔τ相关：适用于小样本
    • 距离相关：衡量非线性相关

回归分析：

  • 简单线性回归：
    • 回归方程：Y = a + bX + ε
    • 最小二乘法：估计回归参数
    • 决定系数R²：解释变异的比例
    • 回归诊断：残差分析、异常值检测
  • 多元线性回归：
    • 多元回归方程：Y = a + b₁X₁ + b₂X₂ + ... + ε
    • 变量选择：前进法、后退法、逐步法
    • 多重共线性：方差膨胀因子VIF
    • 模型评价：AIC、BIC准则

数据表达方法

统计图表：

  • 柱状图：
    • 适用数据：分类数据、离散数据
    • 图表特点：直观比较不同类别的数值
    • 变形类型：堆积柱状图、分组柱状图
    • 制作要点：合理的柱宽、清晰的标签
  • 折线图：
    • 适用数据：时间序列数据、连续数据
    • 图表特点：显示数据的变化趋势
    • 变形类型：多线图、面积图
    • 制作要点：合适的时间间隔、平滑的曲线
  • 散点图：
    • 适用数据：两个连续变量的关系
    • 图表特点：显示变量间的相关性
    • 变形类型：气泡图、三维散点图
    • 制作要点：合适的坐标轴范围、趋势线

专题地图：

  • 统计地图：
    • 等值线图：等高线、等温线、等降水量线
    • 分级统计图：按数值分级着色
    • 点值图：用点的大小表示数值
    • 流向图：用箭头表示流向和流量
  • 分布图：
    • 点状分布图：显示现象的空间分布
    • 面状分布图：显示区域的分布特征
    • 线状分布图：显示线性要素的分布
    • 密度图：显示现象的密度分布

选择题

1. 下列哪种方法最适合收集大范围地表温度数据：

解析：遥感监测可以快速获取大范围的地表温度数据，具有覆盖面广、时效性强的优势。

2. 皮尔逊相关系数的取值范围是：

答案：B

解析：皮尔逊相关系数的取值范围是-1到1，绝对值越大表示相关性越强。

3. 标准差的作用是衡量数据的：

答案：B

解析：标准差是衡量数据离散程度的重要指标，标准差越大表示数据越分散。

4. 等值线图最适合表示：

答案：B

解析：等值线图适合表示连续变化的数据，如温度、降水、海拔等。

5. Z-score标准化的目的是：

答案：A

解析：Z-score标准化将数据转换为均值为0、标准差为1的标准正态分布，消除量纲影响。
