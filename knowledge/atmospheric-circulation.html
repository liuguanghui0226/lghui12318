冷热不均引起大气运动

大气运动是由于地球表面受热不均造成的。太阳辐射的差异导致气温差异，进而引起气压差异，形成大气的水平运动和垂直运动，这是天气变化和气候形成的根本原因。

核心概念

热力环流

冷热不均形成的大气环流

  • 受热上升：热空气密度小上升
  • 冷却下沉：冷空气密度大下沉
  • 水平补偿：形成水平气流
  • 闭合环流：构成完整循环系统

气压系统

大气压力的空间分布

  • 高压系统：空气下沉，天气晴朗
  • 低压系统：空气上升，多云降水
  • 气压梯度：推动空气水平运动
  • 等压线：连接气压相等的点

风的形成

空气的水平运动

  • 气压梯度力：推动空气运动
  • 地转偏向力：改变风向
  • 摩擦力：影响风速和风向
  • 风向风速：反映气压分布

详细解析

大气的受热过程

大气受热过程的基本原理：

  • 能量来源：
    • 太阳辐射是大气运动的根本能量来源
    • 太阳辐射能量巨大且相对稳定
    • 地球接收的太阳辐射约为1.74×10¹⁷瓦
  • 辐射特征：
    • 太阳辐射：短波辐射，波长0.15-4.0微米
    • 地面辐射：长波辐射，波长4.0-120微米
    • 大气辐射：长波辐射，与地面辐射相似

太阳辐射的传递过程：

  • 太阳→大气顶部：
    • 太阳常数：1367W/m²（大气顶部垂直面上的辐射强度）
    • 辐射类型：主要为短波辐射
    • 能量分布：紫外线7%，可见光43%，红外线50%
  • 大气对太阳辐射的作用：
    • 吸收作用：
      • 臭氧吸收紫外线（平流层）
      • 水汽吸收红外线（对流层）
      • CO₂吸收红外线
      • 总吸收量约占19%
    • 反射作用：
      • 云层反射：反射率20-90%
      • 大气分子反射：瑞利散射
      • 总反射量约占6%
    • 散射作用：
      • 分子散射：蓝光散射强，天空呈蓝色
      • 尘埃散射：各波长均匀散射
      • 总散射量约占8%
    • 透射作用：
      • 大部分可见光透过大气
      • 到达地面约占47%
      • 直接辐射和散射辐射
  • 地面对太阳辐射的作用：
    • 吸收：地面吸收约47%的太阳辐射
    • 反射：地面反射率（反照率）因地表类型而异
      • 新雪：80-95%
      • 沙漠：25-30%
      • 森林：10-15%
      • 水面：3-10%
    • 温度变化：地面温度升高，成为大气的主要热源

地面辐射和大气辐射：

  • 地面辐射：
    • 辐射类型：长波辐射（红外辐射）
    • 辐射强度：与地面温度的四次方成正比
    • 辐射方向：向上辐射到大气中
    • 能量传递：地面是大气的主要直接热源
  • 大气辐射：
    • 辐射类型：长波辐射
    • 辐射方向：向上向下双向辐射
    • 大气逆辐射：大气向下的辐射，对地面有保温作用
    • 温室效应：大气对地面长波辐射的吸收和再辐射
  • 辐射平衡：
    • 地球系统：收入的太阳辐射=支出的地球辐射
    • 地面：收入（太阳辐射+大气逆辐射）=支出（地面辐射）
    • 大气：收入（地面辐射+太阳辐射）=支出（大气辐射）
    • 动态平衡：维持地球温度相对稳定

大气受热过程示意图

热力环流的形成

形成过程：

  • 第一步：冷热不均：
    • 地面受热不均匀
    • 形成温度差异
    • 热的地方气温高，冷的地方气温低
  • 第二步：垂直运动：
    • 热空气密度小，上升形成上升气流
    • 冷空气密度大，下沉形成下沉气流
    • 垂直方向上的空气运动
  • 第三步：气压差异：
    • 上升区域地面形成低压
    • 下沉区域地面形成高压
    • 高空气压分布与地面相反
  • 第四步：水平运动：
    • 地面：高压→低压
    • 高空：低压→高压
    • 形成水平补偿气流
  • 第五步：闭合环流：
    • 垂直运动+水平运动
    • 形成完整的环流系统
    • 持续的大气运动

热力环流的特点：

  • 驱动力：温度差异
  • 方向：热处上升，冷处下沉
  • 规模：从局地到全球
  • 时间：可持续可间断

典型实例：

  • 海陆风：
    • 白天：陆地升温快，形成陆地低压，海风
    • 夜晚：陆地降温快，形成陆地高压，陆风
    • 日变化明显
  • 山谷风：
    • 白天：山坡受热，谷风（谷地→山坡）
    • 夜晚：山坡散热，山风（山坡→谷地）
    • 地形影响显著
  • 城市热岛：
    • 城市温度高于郊区
    • 城市上升，郊区下沉
    • 影响城市气候

大气压力与风的形成

气压的概念：

  • 定义：大气对地面或物体表面的压力
  • 单位：百帕（hPa）或毫巴（mb）
  • 标准气压：1013.25hPa（海平面）
  • 测量：气压计测量

气压的影响因素：

  • 海拔高度：
    • 海拔越高，气压越低
    • 每升高100米，气压约降低12hPa
    • 大气密度随高度递减
  • 温度：
    • 温度高，空气上升，地面气压低
    • 温度低，空气下沉，地面气压高
    • 热力作用是主要因素
  • 大气运动：
    • 上升运动形成低压
    • 下沉运动形成高压
    • 动力作用影响气压

风的形成：

  • 气压梯度力：
    • 气压差异产生的力
    • 方向：高压→低压
    • 大小：与气压梯度成正比
    • 是风形成的根本原因
  • 地转偏向力：
    • 地球自转产生的偏向力
    • 北半球向右偏，南半球向左偏
    • 赤道为零，两极最大
    • 改变风向但不改变风速
  • 摩擦力：
    • 地面对空气运动的阻力
    • 减小风速
    • 改变风向（向低压偏转）
    • 高度越高影响越小

风向和风速：

  • 高空风（地转风）：
    • 气压梯度力=地转偏向力
    • 风向与等压线平行
    • 北半球：背风而立，高压在右
  • 近地面风：
    • 受摩擦力影响
    • 风向与等压线斜交
    • 从高压指向低压

全球大气环流概述

形成原因：

  • 根本原因：高低纬度间的热量差异
  • 直接原因：气压带和风带的形成
  • 影响因素：地球自转、海陆分布

理想状态下的环流：

  • 单圈环流模式：
    • 赤道受热上升
    • 极地冷却下沉
    • 高空：赤道→极地
    • 地面：极地→赤道
  • 实际的三圈环流：
    • 哈德里环流（0°-30°）
    • 费雷尔环流（30°-60°）
    • 极地环流（60°-90°）

气压带和风带：

  • 七个气压带：
    • 赤道低气压带
    • 副热带高气压带（南北）
    • 副极地低气压带（南北）
    • 极地高气压带（南北）
  • 六个风带：
    • 信风带（南北）
    • 西风带（南北）
    • 极地东风带（南北）

大气运动的地理意义

热量输送

将低纬度的热量输送到高纬度，调节全球热量分布，缓解温度差异

水汽输送

将海洋上的水汽输送到陆地，形成降水，维持陆地水循环

天气变化

不同性质气团的移动和相遇，形成各种天气现象和天气变化

气候形成

决定各地的气候类型和特征，影响生物分布和人类活动

练习题

选择题

1. 热力环流形成的根本原因是：

答案：B

解析：热力环流形成的根本原因是地面受热不均，导致气温差异，进而引起气压差异和大气运动。

2. 在热力环流中，近地面气压分布规律是：

答案：B

解析：在热力环流中，受热区空气上升形成低压，冷却区空气下沉形成高压。

3. 地转偏向力的特点是：

答案：B

解析：地转偏向力只改变运动物体的方向，不改变运动速度的大小。

4. 海陆风现象中，白天风向是：

答案：B

解析：白天陆地升温快形成低压，海洋升温慢形成高压，风从海洋吹向陆地，称为海风。

5. 大气对太阳辐射的削弱作用不包括：

答案：D

解析：大气对太阳辐射的削弱作用包括吸收、反射、散射三种，辐射作用是大气自身的能量释放。

1. 分析热力环流的形成过程及其地理意义。

解答：
形成过程：
1. 受热不均：地球表面不同地区受热程度不同
2. 气温差异：受热多的地区气温高，受热少的地区气温低
3. 密度差异：热空气密度小，冷空气密度大
4. 垂直运动：热空气上升，冷空气下沉
5. 水平补偿：形成水平气流，构成闭合环流

地理意义：
• 形成海陆风、山谷风等局地环流
• 影响天气变化和气候形成
• 促进大气中热量和水汽的输送
• 影响污染物扩散和空气质量

2. 说明气压梯度力、地转偏向力、摩擦力对风向风速的影响。

解答：
气压梯度力：
• 方向：垂直于等压线，从高压指向低压
• 作用：推动空气运动，是风的原动力
• 大小：与气压梯度成正比

地转偏向力：
• 方向：北半球向右偏，南半球向左偏
• 作用：改变风向，不改变风速
• 大小：与风速和纬度正弦成正比

摩擦力：
• 方向：与风向相反
• 作用：减小风速，改变风向
• 影响：近地面明显，高空较小

3. 分析海陆风、山谷风的形成机制。

解答：
海陆风：
• 海风：白天，陆地升温快，形成低压；海洋升温慢，形成高压；风从海洋吹向陆地
• 陆风：夜晚，陆地降温快，形成高压；海洋降温慢，形成低压；风从陆地吹向海洋

山谷风：
• 谷风：白天，山坡升温快，空气上升；山谷空气沿山坡上升
• 山风：夜晚，山坡降温快，空气下沉；山谷空气沿山坡下沉

共同特点：
• 都是局地热力环流
• 具有日变化规律
• 影响局地气候和天气

案例分析

城市热岛效应与大气环流

现象描述：城市中心气温明显高于郊区，形成"热岛"现象。

形成原因：
• 人为热源：工业、交通、建筑等人为活动释放大量热量
• 下垫面差异：城市建筑、道路等人工下垫面吸热能力强
• 植被减少：城市植被覆盖率低，蒸腾作用减弱
• 空气污染：污染物吸收和反射太阳辐射

环流影响：
• 城市中心形成低压，郊区形成高压
• 产生从郊区向城市中心的"城市风"
• 影响污染物扩散和空气质量
• 加剧城市热岛效应

缓解措施：
• 增加城市绿地和水体
• 改善建筑设计和材料
• 发展清洁能源和绿色交通
• 合理规划城市布局

重要概念

热力环流

由于地表受热不均而形成的大气环流，是大气运动的基本形式。

气压梯度力

由于气压差异而产生的力，是推动空气水平运动的原动力。

地转偏向力

由于地球自转产生的偏向力，北半球向右偏，南半球向左偏。

海陆风

由于海陆热力性质差异形成的局地环流，具有明显的日变化。
