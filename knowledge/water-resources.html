水资源的合理利用

水资源是人类生存和发展的基础资源，合理开发利用和保护水资源对于可持续发展具有重要意义。了解水资源的分布特点、开发利用现状和保护措施是地理学习的重要内容。

核心概念

水资源概况

全球和中国水资源分布特点

  • 总量丰富：全球水资源总量大
  • 淡水稀少：可利用淡水资源有限
  • 分布不均：时空分布极不均匀
  • 供需矛盾：需求增长与资源有限

水资源问题

当前面临的主要水资源问题

  • 水资源短缺：供不应求
  • 水质污染：水体污染严重
  • 浪费严重：利用效率低
  • 分配不均：区域差异明显

合理利用

水资源的可持续利用途径

  • 开源：开发新水源
  • 节流：提高用水效率
  • 保护：防治水污染
  • 管理：科学配置水资源

详细解析

水资源的分布特点

全球水资源概况：

  • 水资源总量：
    • 地球表面约71%被水覆盖
    • 总水量约13.86亿立方千米
    • 海水占97.5%，淡水仅占2.5%
    • 可利用淡水不足总量的1%
  • 淡水资源构成：
    • 冰川和永久积雪：68.7%
    • 地下水：30.1%
    • 地表水：1.2%（湖泊、河流、土壤水、大气水等）
  • 可利用水资源：
    • 河流水：约0.0002%
    • 淡水湖泊：约0.007%
    • 浅层地下水：约0.3%
    • 土壤水分：约0.001%

全球水资源分布特点：

  • 地区分布不均：
    • 亚洲：拥有全球36%的水资源，60%的人口
    • 南美洲：拥有26%的水资源，6%的人口
    • 北美洲：拥有15%的水资源，8%的人口
    • 非洲：拥有11%的水资源，13%的人口
    • 欧洲：拥有7%的水资源，12%的人口
    • 大洋洲：拥有5%的水资源，1%的人口
  • 气候带分布：
    • 热带雨林气候区：水资源丰富
    • 温带海洋性气候区：水资源较丰富
    • 干旱半干旱气候区：水资源稀缺
    • 极地气候区：水资源以冰雪形式存在
  • 季节分布不均：
    • 季风气候区：雨季集中，旱季缺水
    • 地中海气候区：冬雨夏干
    • 温带大陆性气候区：春季融雪期水量大

中国水资源分布特点：

  • 总量特征：
    • 水资源总量约2.8万亿立方米
    • 人均水资源量约2100立方米
    • 仅为世界平均水平的1/4
    • 属于水资源相对短缺国家
  • 空间分布：
    • 南多北少：南方占81%，北方占19%
    • 东多西少：东部沿海水资源相对丰富
    • 山区多平原少：山区水资源集中
    • 胡焕庸线：东南部人多水少，西北部人少水也少
  • 时间分布：
    • 夏秋多冬春少：与降水季节分布一致
    • 年际变化大：丰水年与枯水年差异明显
    • 洪枯交替：洪水期与枯水期交替出现

全球水资源概况：

  • 总量：
    • 地球水体总量：约13.86亿km³
    • 海水：96.5%
    • 淡水：2.5%
    • 可利用淡水：不到1%
  • 淡水分布：
    • 冰川和永久积雪：68.7%
    • 地下水：30.1%
    • 地表水：1.2%
    • 其中河流水仅占0.0002%

全球水资源空间分布：

  • 丰富地区：
    • 亚马逊河流域：世界最大
    • 刚果河流域：非洲最大
    • 北美五大湖地区
    • 俄罗斯西伯利亚地区
  • 贫乏地区：
    • 中东地区：极度缺水
    • 北非地区：沙漠气候
    • 澳大利亚内陆：干旱气候
    • 美国西南部：半干旱气候

中国水资源特点：

  • 总量：
    • 年均降水量：约6.2万亿m³
    • 水资源总量：约2.8万亿m³
    • 人均水资源：约2100m³（世界平均1/4）
  • 空间分布：
    • 南多北少：南方占80%，北方占20%
    • 东多西少：东部湿润，西部干旱
    • 山区多平原少：地形影响明显
  • 时间分布：
    • 夏秋多冬春少：季节分配不均
    • 年际变化大：丰水年与枯水年差异大
    • 与农业用水需求不匹配

主要水资源问题

水资源短缺：

  • 资源性缺水：
    • 定义：水资源总量不足
    • 原因：降水少，蒸发大
    • 分布：西北干旱半干旱地区
    • 特点：绝对缺水
  • 工程性缺水：
    • 定义：缺乏水利工程设施
    • 原因：投资不足，技术落后
    • 分布：西南山区
    • 特点：有水用不上
  • 水质性缺水：
    • 定义：水质污染严重
    • 原因：工业污染，生活污水
    • 分布：经济发达地区
    • 特点：有水不能用

水污染问题：

  • 污染源：
    • 工业污染：重金属、化学物质
    • 农业污染：化肥、农药
    • 生活污染：生活污水、垃圾
    • 其他污染：船舶、大气沉降
  • 污染类型：
    • 化学污染：有毒有害物质
    • 生物污染：病原微生物
    • 物理污染：悬浮物、热污染
  • 影响：
    • 饮用水安全：威胁人体健康
    • 生态破坏：水生生物死亡
    • 经济损失：治理成本高

用水浪费：

  • 农业用水：
    • 灌溉方式落后：大水漫灌
    • 输水损失大：渠道渗漏
    • 用水效率低：单位产出耗水多
  • 工业用水：
    • 技术落后：单位产值耗水高
    • 循环利用率低：重复利用少
    • 节水意识差：管理粗放
  • 生活用水：
    • 节水意识不强：浪费现象普遍
    • 设施老化：管网漏损严重
    • 用水结构不合理：优质水劣用

水资源的开发利用

开源措施：

  • 地表水开发：
    • 修建水库：调节径流，蓄水供水
    • 引水工程：跨流域调水
    • 河流整治：提高河道输水能力
    • 洪水利用：洪水资源化
  • 地下水开发：
    • 浅层地下水：打井取水
    • 深层地下水：深井开采
    • 地下水回灌：人工补给
    • 注意：防止过度开采
  • 非常规水源：
    • 海水淡化：膜法、热法
    • 污水处理回用：中水利用
    • 雨水收集：城市雨水利用
    • 空中水资源：人工增雨

节流措施：

  • 农业节水：
    • 节水灌溉：滴灌、喷灌、微灌
    • 渠道防渗：减少输水损失
    • 作物改良：选育抗旱品种
    • 农艺节水：覆盖保墒、深耕
  • 工业节水：
    • 技术改造：采用节水工艺
    • 循环利用：提高重复利用率
    • 分质供水：优质水优用
    • 废水处理：达标排放或回用
  • 生活节水：
    • 节水器具：节水龙头、马桶
    • 分质供水：生活用水分级
    • 中水利用：处理后再利用
    • 宣传教育：提高节水意识

水资源保护：

  • 污染防治：
    • 源头控制：清洁生产
    • 过程控制：污水处理
    • 末端治理：达标排放
    • 生态修复：水体自净
  • 生态保护：
    • 水源地保护：划定保护区
    • 湿地保护：维护生态功能
    • 河流生态：保持河流连通性
    • 流域管理：统筹上下游

典型案例分析

南水北调工程：

  • 背景：
    • 北方缺水严重
    • 南方水资源相对丰富
    • 经济发展需要
  • 工程概况：
    • 东线：长江→黄河→海河→京津
    • 中线：丹江口→北京、天津
    • 西线：长江上游→黄河上游
  • 意义：
    • 缓解北方缺水
    • 促进经济发展
    • 改善生态环境
  • 问题：
    • 投资巨大
    • 生态影响
    • 水质保护

以色列节水农业：

  • 背景：
    • 干旱半干旱气候

    • 农业发展需求
  • 措施：
    • 滴灌技术：精准灌溉
    • 海水淡化：增加水源
    • 污水回用：循环利用
    • 科学管理：精细化管理
  • 效果：
    • 农业用水效率世界领先
    • 沙漠变绿洲
    • 农产品出口

新加坡水资源管理：

  • 四大水源：
    • 本地集水：雨水收集
    • 进口水：从马来西亚进口
    • 新生水：污水深度处理
    • 海水淡化：反渗透技术
  • 管理特点：
    • 统一管理：一个部门负责

水资源利用模式

传统模式

开发→利用→排放的线性模式，资源消耗大，环境污染重

节约模式

提高用水效率，减少浪费，但仍是线性思维

循环模式

污水处理回用，形成循环利用，减少新水需求

生态模式

人水和谐，生态优先，可持续利用水资源

练习题

选择题

1. 全球淡水资源中，可直接利用的水资源主要是：

答案：C

解析：虽然冰川水和地下水占淡水资源的大部分，但可直接利用的主要是河流水和淡水湖泊水，它们易于开发利用且更新速度快。

2. 中国水资源分布的基本特点是：

答案：B

解析：中国水资源分布的基本特点是南多北少，南方地区拥有全国81%的水资源，而北方地区仅占19%，这与降水分布密切相关。

3. 解决水资源短缺问题的根本途径是：

答案：C

解析：节约用水、提高水资源利用效率是解决水资源短缺问题的根本途径，其他措施都是辅助手段，且成本较高或有局限性。

4. 南水北调工程主要解决的问题是：

答案：C

解析：南水北调工程是将南方丰富的水资源调到缺水的北方地区，主要解决水资源空间分布不均的问题。

5. 农业节水的主要技术措施是：

答案：B

解析：滴灌和喷灌是现代农业节水的主要技术措施，能够精确控制用水量，大幅提高水资源利用效率。

综合分析题

6. 阅读材料，回答问题：

华北地区是中国重要的农业区和工业区，人口稠密，经济发达。但该地区水资源严重短缺，人均水资源量仅为全国平均水平的1/7。近年来，华北地区地下水位持续下降，形成了世界上最大的地下水漏斗区。为缓解华北地区水资源短缺问题，国家实施了南水北调工程。

问题：

（1）分析华北地区水资源短缺的原因。（8分）

（2）说明地下水过度开采的危害。（6分）

（3）除南水北调外，提出华北地区解决水资源短缺的其他措施。（6分）

答案要点：

（1）华北地区水资源短缺的原因：

  • 自然原因：
    • 降水量少：年降水量400-800mm，且集中在夏季
    • 蒸发量大：气候干燥，蒸发强烈
    • 河流径流量小：河流水量有限，且季节变化大
  • 人为原因：
    • 人口稠密：用水需求量大
    • 经济发达：工农业用水量大
    • 水资源浪费：利用效率低，浪费严重
    • 水污染严重：可利用水资源减少

（2）地下水过度开采的危害：

  • 地面沉降：地下水位下降导致地面塌陷
  • 地下水漏斗：形成大面积地下水位下降区
  • 水质恶化：深层地下水水质较差，含盐量高
  • 生态破坏：湿地干涸，植被退化
  • 海水入侵：沿海地区地下水位下降导致海水倒灌
  • 建筑物损坏：地面沉降影响建筑物安全

（3）其他解决措施：

  • 节约用水：推广节水技术，提高用水效率
  • 污水处理：加强污水处理，实现中水回用
  • 雨水收集：建设雨水收集系统，增加水源
  • 海水淡化：沿海地区发展海水淡化技术
  • 产业调整：调整产业结构，发展节水型产业
  • 水价改革：实行阶梯水价，促进节约用水
